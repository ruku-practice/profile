# 自己紹介Webサイト（ポートフォリオ/リンク集）仕様書

## 1. 目的とコンセプト
lit.link（リットリンク）のような、自身の活動やSNSリンクをまとめたシンプルな自己紹介Webサイトを作成する。
訪問者が一目で「誰で」「何をしているのか」を把握でき、各メディア（X、Substackなど）へスムーズに遷移できるようにする。

## 2. ターゲットユーザー
- X（Twitter）やSubstackのフォロワー
- スペース配信のリスナー
- AI活用やアプリ開発に興味がある人

## 3. 構成と機能要件
サイトは1ページ（シングルページ）構成とする。

### 3.1. プロフィールセクション
X（Twitter）のプロフィール情報を元に作成。

- **名前:** ルク｜AI活用で人生の踊り場を突破
- **アイコン画像:** （※画像の準備が必要です）
- **自己紹介テキスト（案）:**
  会社員としてキャリアの踊り場に立ち、AIを活用して次のステップを模索中。
  AIバイブコーディングを用いて習慣化アプリを開発・リリースした経験を持ちます。
  SNS運用や日常生活をAIで自動化するプロセスを全公開しており、平日朝8時頃・土日朝7時にはXのスペースで配信を行っています！

### 3.2. リンク集セクション
以下のプラットフォームへのリンクボタンを設置する。
- **X (Twitter):** https://x.com/ruku_practice
- **Substack:** https://substack.com/@rukupractice
- **開発したアプリ:** （※URLがあれば追加）

### 3.3. コンテンツセクション（拡張案・任意）
- 開発した習慣化アプリへのリンク・紹介
- 最新のSubstack記事リスト（手動更新）
- 配信（スペース）の案内

## 4. 今後決めるべきこと・質問事項（★ルクさんへ）
方向性を決めるために、以下の点について検討・回答をお願いします。

1. **デザインの雰囲気（テーマ）**
   - どのようなデザインが好みですか？
   - 例: 「シンプルでモダン（白黒基調）」「温かみのあるデザイン」「AIらしさのある少しサイバーな感じ」など。
   A シンプルでモダン（白黒基調）でお願いします
2. **テーマカラー**
   - ベースとなる色や、ボタンの色などの希望はありますか？
   A ブルーグリーンが好きです。この色をベースに作ってもらえたらと思います。
3. **掲載するリンクについて**
   - X、Substack以外に載せたいリンク（開発したアプリのURL、GitHub、noteなど）はありますか？
   A 縦読みリーダーリンクぐらいでいいです。URLわかるでしょうか？

4. **ホスティング（公開方法）について**
   - 完成したWebサイトはどこで公開する予定ですか？
   - （例: Vercel, Netlify, GitHub Pages など。無料で簡単に公開できる方法も提案可能です）
   A 無料のところでおすすめを教えてもらえると幸いです。今まで触ったことがあるものは、vereclとGithubあたりです。すぐに使えそうなものがあれば、それを使ってもらえらばOKです。
5. **技術スタック**
   - シンプルな HTML/CSS/JS で作るか、React (Next.js/Vite) などのフレームワークを使って拡張性を持たせるか、どちらが良いでしょうか？（今後の拡張予定がなければHTML/CSSで十分です）
   A 今回はシンプルなもので良いかと思います。

   あとプロファイルアイコンは、Xのものを流用してください。
